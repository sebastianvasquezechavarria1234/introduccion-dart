# Dart Introduction Project

A console-based application developed in Dart that provides a collection of utility calculations and exercises. This project focuses on clean code practices, modular design, and robust user input handling.

## 🚀 Features

The application includes a menu-driven interface with the following functionalities:

1.  **Physics Calculations**: Calculate distance based on speed and time.
2.  **Academic Tools**: Compute the average of three grades.
3.  **Sports Statistics**: Calculate total points for a soccer team based on wins, draws, and losses.
4.  **Payroll Management**: Generate employee payroll information based on hours worked and hourly rates.
5.  **Geometry**: Calculate the hypotenuse of a right triangle using the Pythagorean theorem.
6.  **Unit Conversion**: Convert temperatures from Celsius to Fahrenheit.

## 🛠️ Technical Improvements

This project has been refactored to follow professional standards:

*   **Modular Architecture**: Logic is separated into `main.dart` for the UI/Flow and `operaciones.dart` for core calculations.
*   **Robust Input Validation**: Custom handlers to manage data types and prevent application crashes from invalid user input.
*   **Null Safety**: Fully compliant with Dart's null safety features.
*   **Enhanced UX**: Includes console screen clearing and interactive pauses for a better terminal experience.
*   **Documentation**: All functions are documented using Dart's documentation comments (`///`).

## 💻 Prerequisites

*   [Dart SDK](https://dart.dev/get-dart) installed on your system.

## 🏃 How to Run

1.  Clone the repository:
    ```bash
    git clone https://github.com/sebastianvasquezechavarria1234/introduccion-dart.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd introduccion-dart
    ```
3.  Run the application:
    ```bash
    dart main.dart
    ```
